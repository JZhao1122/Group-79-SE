üöÄ Getting Started

Quick Setup

# 1. Setup JUnit Environment setup_junit.bat # 2. Compile Project compile_project.bat # 3. Run Tests run_tests.bat # 4. Launch Application java -cp "out/production/Group-79-SE-main;lib/*" ui.DeepManageApp

Example Usage

// Create and analyze transactions List<Transaction> transactions = Arrays.asList( new Transaction(LocalDate.now(), 45.0, "Dining", "Starbucks", "Credit Card"), new Transaction(LocalDate.now(), 300.0, "Transportation", "Gas Station", "Debit Card") ); // Train AI classifier AITransactionClassifier.trainModel(transactions); // Get budget recommendations List<BudgetRecommender.BudgetSuggestion> suggestions = BudgetRecommender.recommendBudget(transactions); // Create savings plan SavingsOptimizer.SavingsPlan plan = SavingsOptimizer.generatePlan( 10000.0, LocalDate.now().plusMonths(6), transactions);
